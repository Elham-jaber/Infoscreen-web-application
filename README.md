# InfoScreen — Boulangerie

##  Présentation
**InfoScreen Boulangerie** est une application web développée en **PHP**
permettant l’**affichage d’actualités et d’informations** sur un écran destiné
aux clients d’une boulangerie.

L’objectif est de diffuser facilement :
- les actualités (nouveaux produits, promotions),
- les horaires et informations importantes,
- des messages visibles sur un écran public.

Le projet utilise une **base de données MariaDB** et des **requêtes SQL**,
sans framework.

---

## Technologies utilisées
- PHP
- MariaDB
- SQL
- phpMyAdmin
- HTML
- CSS
- Bootsrap (Framework CSS)
- JavaScript

---

##  Fonctionnalités
- Affichage dynamique des actualités depuis la base de données
- Gestion des sessions utilisateurs
- Ajout d’informations via des formulaires
- Séparation claire entre affichage et logique métier
- Interface adaptée à un écran d’information

---

## Organisation du projet

- `index.php` : point d’entrée de l’application
- `affichage/` : récupération et affichage des actualités
- `inscription/` : formulaires et insertion des données
- `session/` : gestion des sessions et sécurité
- `css/` : styles de l’application
- `js/` : scripts JavaScript
- `assets/` : images et ressources

---

##  Base de données
Les données sont stockées dans une base **MariaDB**.
